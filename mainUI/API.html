declarationForm API documentation version 1

http://localhost:8080/{version}

  • version: required (1)

概述

declarationFrom是一个使用微服务架构开发的针对海关报关、通关业务场景的演示系统，旨在展示FusionStage平台微服务相关的各项特性。 系统特点：

  • 所有微服务皆使用ServiceComb框架
  • 事务一致性使用ServiceComb Saga
  • 用户界面Vue+ElementUI
  • 数据均使用Mysql5.7
  • 所有微服务均部署为无状态容器
